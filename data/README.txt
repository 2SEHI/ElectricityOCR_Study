1. 전력량계량기와 모뎀 파일명의 매칭 기준 : 
전력량계량기는 _P1이 포함되고 모뎀 파일명은 _P2가 포함됨
파일명의 "_P1" 과 "_P2" 이외의 문자열이 일치함
전력량계량기에 언더바기준으로 앞의 숫자가 같은 파일이 여러개 존재하기도 함

ex)
	전력량계량기 : XXXX_P1XXXX.jpg 
	모뎀       : XXXX_P2XXXX.jpg 


2. 모뎀 또는 전력량계량기에만 존재하는 파일
	모뎀에만 존재하는 파일 수 : 187 장
	전력량계량기에만 존재하는 파일 수 : 187 장
     (서로 파일의 개수는 같으나 파일명은 모두 다름)
